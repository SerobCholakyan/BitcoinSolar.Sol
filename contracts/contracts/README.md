# BLSR Smart Contract Suite

## Deployment Instructions
1. Upload all three `.sol` files to Remix.
2. Compile `BitcoinSolar.sol`.
3. Deploy to Polygon Mainnet.

## Security Features
- **Capped Supply:** Hard-coded at 21,000,000.
- **Node Authorization:** Only your specific backend wallet can trigger mining.
- **Bit-Shift Math:** Uses assembly-level efficiency for halving rewards.
